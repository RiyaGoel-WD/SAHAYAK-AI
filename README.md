# 🛡️ Sahayak AI
### Your Personal Safety Companion

Sahayak AI is an all-in-one emergency safety platform built for everyone.
Designed to make emergency response faster and smarter.

---

## 🚀 Features

### 🆘 1. One-Tap SOS Emergency
- Select your emergency type (Medical / Crime / Cyber)
- Auto-fetches your live location
- Sends alert message to multiple emergency contacts
- Connects to nearest help finder

### 🔍 2. Cyber Fraud Detection
- Paste any suspicious message or scam text
- AI analyses it instantly
- Shows verdict + suggested actions to take

### 📍 3. Nearest Help Locator
- Detects your situation type
- Finds nearest Hospital, Police Station, or Cyber Cell
- Shows directions and contact info

---

## 🛠️ Built With
- HTML, CSS, JavaScript
- Claude AI API (for fraud detection)
- Browser Geolocation API (for live location)
